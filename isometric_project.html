Isometric Shooter Project

This third-person shooter game features an isometric perspective. It can be played by clikcing on the picture below! In this game you are having a dream that the toys in your room have become zombies, and are coming to get you! This game is largely adapted from Unity tutorials, although I have made changes in controls, enemy management, and UI.

If you are using the Chrome Browser, please be aware that Chrome no longer supports the Unity Web Player. It should work on Firefox or Internet Explorer.
