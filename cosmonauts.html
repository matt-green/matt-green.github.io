Super Cosmonauts

Super Cosmonauts is a top-down arcade style space shooter. It can be played by clikcing on the picture below! The game has 4 levels, and a final boss at the end. Can you make it through all 4 levels? If you are using the Chrome Browser, please be aware that Chrome no longer supports the Unity Web Player. It should work on Firefox or Internet Explorer.
